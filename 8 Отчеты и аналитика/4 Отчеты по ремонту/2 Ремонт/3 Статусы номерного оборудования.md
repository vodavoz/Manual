# Статусы номерного оборудования

## Описание отчета

Отчет выводит оборудование с любым типом, статусом и местоположением.

Этот отчёт является основой для:

- Полная инвентаризация: предоставляет полный список всего номерного оборудования, находящегося за пределами склада компании.

- Отслеживание местонахождения оборудования: указывает, у какого клиента, по какому адресу и находится каждая единица оборудования.

- Анализ использования оборудования: позволяет понять, какое оборудование находится у клиента, в ремонте или на складе.

- Идентификация и учёт: для каждой единицы оборудования указывается его статус, владелец, серийный номер (S/N), а также адрес, по которому оно находится.

## Виды отчета

Отчет находится находится [!badge variant="primary" text="Ремонт и аренда"]->[!badge variant="success" text="Отчеты"]->[!badge variant="light" text="Отчеты по ремонту и аренде"]->[!badge variant="success" text="Ремонт"]->[!badge variant="light" text="Статусы номерного оборудования"].

Отчет имеет следующие варианты вывода:

+++ На складах

Отчет формируется только по оборудованию имеющему статус [!badge variant="info" text="На складе"].

![](/images/Отчет_статусы_оборудования_склад.jpg)

Данные в отчете сгруппированы по типу оборудования и месту нахождения (складу).

[!badge variant="info" text="Владелец оборудования"] если выбран владелец оборудования, то в отчет попадет только его личное оборудование. 

[!badge variant="info" text="Статус оборудования"] если выбран статус, то в отчет будет сформирован только выбранному статусу.

В колонках выводится наименование оборудования, контрагент, которому оно принадлежит и количество.

+++ Оборудование клиента

Основной вид отчета. Данные в отчете сгруппированы по статусу оборудования.

![](/images/Отчет_статусы_оборудования_клиентское.jpg)

[!badge variant="info" text="Владелец оборудования"] если выбран владелец оборудования, то в отчет попадет только его личное оборудование. 

[!badge variant="info" text="Статус оборудования"] если выбран статус, то в отчет будет сформирован только выбранному статусу.

В колонках выводится контрагент, которому принадлежит оборудование, наименование, адрес доставки КА, склад, на котором оно находится и количество.

+++ У клиента

Отчет формируется только по оборудованию имеющему статус [!badge variant="info" text="У клиента"].

![](/images/Отчет_статусы_оборудования_у_клиента.jpg)

Данные в отчете сгруппированы по типу оборудования.

[!badge variant="info" text="Владелец оборудования"] если выбран владелец оборудования, то в отчет попадет только его личное оборудование. 

[!badge variant="info" text="Статус оборудования"] если выбран статус, то в отчет будет сформирован только выбранному статусу.

В колонках выводится наименование оборудования, контрагента, которому оно принадлежит, статус, адрес доставки КА и количество.
+++
