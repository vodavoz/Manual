# Отчет по заказ-нарядам

## Описание отчета

Отчет выводит заказ-наряды за определенный период времени с указанием их суммы и суммы выплат мастерам по ним.

- Детализация по клиентам: группирует заказы по контрагентам, позволяя просмотреть всю историю заказов для каждого клиента.

- Анализ выполненных работ/услуг: внутри каждого заказа-наряда указана подробная информация о выполненных работах или услугах (например, «Диагностика», «Санитарная обработка кулера», «Доставка»).

- Учет сумм: показывает общую сумму заказа-наряда.

- Анализ работы мастеров: позволяет в дальнейшем анализировать работу мастеров, выполняющих заказы.

- Формирование отчетности: может использоваться для внутренней отчетности, анализа продаж, оценки эффективности работы сервисной службы.

## Вид отчета

Отчет находится находится [!badge variant="primary" text="Ремонт и аренда"]->[!badge variant="success" text="Отчеты"]->[!badge variant="light" text="Отчеты по ремонту и аренде"]->[!badge variant="success" text="Ремонт"]->[!badge variant="light" text="Отчет по заказ-нарядам"].

После формирования, отчет имеет следующий вид:

![Отчет по заказ-нарядам](/images/Отчет_по_заказ_нарядам.jpg)

Данные в отчете сгруппированы по контрагентам и документам (заказ-нарядам).

[!badge variant="info" text="Организация"] если выбрана организация, то в отчет попадут все контрагенты, у которых заключен договор Ремонт оборудования с указанной организацией. 

[!badge variant="info" text="Мастер"] если выбран мастер, то в отчет попадут все заказ-наряды, отработанные указанным мастером.

В колонках выводится наименование номенклатуры (работ, услуг), сумма по заказ-наряду, сумма выплаты мастеру, который выполнил работы.

!!!Для расчета суммы выплат в номенклатуре должны быть проставлены проценты оплаты.
!!!

## Права доступа

Отчет доступен пользователям, которым назначена одна из ролей: [!badge variant="info" text="Мастер"], [!badge variant="info" text="Работа с ремонтом и арендой"].
