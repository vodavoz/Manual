# Заказы по СБП

## Описание отчета

Отчет предназначен для просмотра статусов оплаты заказов, ведения статистики по суммам, выставленных заказов относительно сумм оплаты.

- Отслеживание статуса заказов: позволяет видеть, какие заказы ожидают оплаты, а какие уже оплачены.

- Ведение статистики по суммам выставленных заказов: в отчете собрана информация о суммах всех выставленных заказов.

- Сравнение сумм выставленных заказов с суммами оплаты: это позволяет увидеть, насколько полно и своевременно поступают платежи по выставленным счетам, а также выявить возможные расхождения или задержки.

Это ключевой инструмент для контроля финансовых потоков, связанных с заказами, оплаченными через Систему быстрых платежей, а также для оценки эффективности процесса продаж и оплаты.
