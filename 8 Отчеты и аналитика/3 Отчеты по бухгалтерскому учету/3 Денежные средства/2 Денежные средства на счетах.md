# Денежные средства на счетах

## Описание отчета

Отчет показывает движение безналичных денежных средств за выбранный период.

Вот основные цели этого отчета:

- Контроль остатков на банковских счетах: показывает, сколько денег находится на каждом банковском счёте организации на начало и конец периода.

- Учет поступлений (приход) и выплат (расход) по банковским счетам: фиксирует все безналичные операции — переводы, платежи, поступления от клиентов, выплаты поставщикам и т. д.

- Детализация операций: показывает, по какому документу (например, «Поступление на расчетный счет № …») произошло движение средств, на какой счет и за какой период.

- Сверка с банковскими выписками: отчет помогает сверить данные системы с банковскими выписками, чтобы убедиться в точности учета.

Этот отчёт является основополагающим для финансового отдела, бухгалтерии и руководства компании в вопросах управления денежными средствами на банковских счетах.
