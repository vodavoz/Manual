# Изменения в версии 2.26.0.0

# Общие изменения 

Добавлены новые печатные формы для [!badge variant="light" text="УПД"] и [!badge variant="light" text="Счет-фактуры"] с поддержкой графы [!badge variant="light" text="5б"].

Обновлен [!badge variant="light" text="Модуль ЭДО"] до версии 1.9.15.100, добавлена поддержка графы 5б.

Добавлена поддержка [!badge variant="light" text="Авансовых счетов фактур"].

Добавлена обработка для массового формирования авансовых СФ по данным остаткам, а так же с возможностью дальнейшей выгрузки в бухгалтерию предприятия. 

Внешний вид многих форм документов возвращен в привычный формат, изменены параметры открытия окон форм, как в предыдущих версиях. 

Обновлена подсистема маркировки в связи с внедрением [!badge variant="light" text="Модуля ПИоТ"].

Улучшено [!badge variant="light" text="Рабочее место кассира"] и добавлен функционал разделения чеков по кассам.

Изменена логика формирования СФ по актам услуг. Теперь документ формируется автоматически в случае, если по акту есть НДС. Как печатная форма выходит УПД в случаях если она указана в адресе доставки. 

Улучшена работа с обработкой оригиналов бухгалтеских документов. Доработан интерфейс.

# Исправление

При перезаполнении реквизитов с помощью 1с контрагент теперь корректно записываются данные.

Исправлены ошибки связанные с переходом на 22% НДС по ЭДО.

Исправлена ошибка передачи статусов маркирвки при работе с ЭДО.

Оптимизирован рассчет ЗП водителям, повышена производительность.

Исправлены ошибки в механизме формирования комплекта документов при перезаписи и добавлении в очередь.
