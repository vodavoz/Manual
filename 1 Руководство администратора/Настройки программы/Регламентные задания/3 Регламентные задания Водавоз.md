# Регламентные задание Водавоз

## Архивирование документов

Регламентное задание 1С предназначено для автоматического перемещения редко используемых или завершённых документов из основной базы данных в архив. 
Данный процесс позволяет освободить основное хранилище от старого материала, увеличить производительность системы и упорядочить хранение информации.

Стандартное расписание:

[!badge variant="info" text="каждый день, по дням недели [понедельник, вторник, среда, четверг, пятница, суббота]; с 18:00:00 по 20:00:00 каждые 300 секунд"].

## Обработка отчетов мобильного приложения

Регламентное задание 1С предназначено для автоматической обработки и анализа данных, поступающих из мобильных устройств, оснащенных специализированным приложением для работы с системой 1С. 
Основное назначение задания — импорт данных, собранных мобильными устройствами, их последующая обработка и интеграция в основную информационную базу предприятия.

Стандартное расписание:

[!badge variant="info" text="каждый день, по дням недели [понедельник, вторник, среда, четверг, пятница, суббота]; с 8:00:00 по 20:00:00 каждые 180 секунд, повторять после завершения через 30 секунд"].

## Пролонгация аренды

Регламентное задание 1С служит для автоматического продления договоров аренды имущества или оборудования, заключенных организацией.
Это задание направлено на автоматизацию процесса пролонгации соглашений, уменьшение ручного труда и предупреждение просрочек арендных обязательств.

Стандартное расписание:

[!badge variant="info" text="каждый день; с 3:00:00 по 4:00:00 каждые 300 секунд"].

## Формирование push уведомлений по динамике потребления

Регламентное задание 1С предназначено для автоматического создания push-уведомлений, содержащих информацию о потреблении воды. 
Работает как напоминание клиентам, которые пользуются мобильным приложением, что пора делать заказ.

Стандартное расписание:

[!badge variant="info" text="каждый день; с 19:00:00 по 20:00:00 каждые 300 секунд, повторять после завершения через 300 секунд"].

## Формирование аналитики звонок заказ

Регламентное задание 1С предназначено для автоматического анализа взаимосвязи звонков и последующих заказов клиентов. 
Основной целью задания является оценка эффективности маркетинговых каналов, кол-центров и менеджеров по привлечению клиентов.
Предназначен для отчета "Статистика эффективности звонков".

Стандартное расписание:

[!badge variant="info" text="каждый день; с 8:00:00 по 22:00:00 каждые 300 секунд"].
