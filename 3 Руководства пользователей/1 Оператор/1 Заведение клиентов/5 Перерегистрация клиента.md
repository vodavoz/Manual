# Перерегистрация клиента

Это процесс изменения данных клиента, связанных с конкретными документами. Это может потребоваться в случае изменения реквизитов клиента, например, при смене юридического адреса, наименования или других важных данных. 

## Форма элемента

![](\images\оператор\клиент.jpg)

[!badge size="xl" variant="secondary" text="Дата"] - устанавливается текущая дата.

[!badge size="xl" variant="secondary" text="Номер"] - присваивается системой автоматически после проведения документа.

[!badge size="xl" variant="secondary" text="Организаия изначальная"] – указывается организация, от имени которой работал контрагент.

[!badge size="xl" variant="secondary" text="Контрагент изначальный"] - указывается контрагент, которого хотите перерегистрировать.

[!badge size="xl" variant="secondary" text="Договор изначальный"] - заполняется договором, на основании которого работал контрагент. 

[!badge size="xl" variant="secondary" text="Организаия новая"] – указывается организация, от имени которой будет работать контрагент.

[!badge size="xl" variant="secondary" text="Контрагент новый"] - указывается контрагент, на которого хотите переименовать.

[!badge size="xl" variant="secondary" text="Договор новый"] - заполняется договором, на основании которого будет работать контрагент.

### Описание вкладок формы элемента

+++ Адреса

На вкладке добавляются адреса, которые необходимо перезакрепить.

![](\images\оператор\клиент1.jpg)

Адреса подтягиваются автоматически, после выбора контрагента в [!badge size="xl" variant="secondary" text="Контрагент изначальный"].

В столбце [!badge size="xl" variant="secondary" text="Обрабатывать"] выбираем адреса для переноса.

Для переноса адреса нужно выбрать нажать кнопку [!badge size="xl" variant="secondary" text="Перенести адреса"]. Перенесенные адреса отразятся в столбике [!badge size="xl" variant="secondary" text="Адрес доставки новый"].

![](\images\оператор\клиент2.jpg)

+++ Корректировки

На данной вкладке переносятся тара, деньги, залоги.

![](\images\администратор\клиент2.jpg)

Для переноса тары нажимаем на кнопку [!badge size="xl" variant="secondary" text="Перенести тару"].

Для переноса денежных средств нажимаем на кнопку [!badge size="xl" variant="secondary" text="Перенести деньги"].

Для переноса залоговой тары нажимаем на кнопку [!badge size="xl" variant="secondary" text="Перенести залоги"].

После нажатия кнопок создадутся документы корректировки.

![](\images\оператор\клиент3.jpg)

+++ Аренда

Если у клиента была оформлена аренда, то нужно воспользоваться данной вкладкой.

![](\images\администратор\клиент3.jpg)

Для переноса аренды нажимаем на кнопку [!badge size="xl" variant="secondary" text="Перенести аренду"].

Создадутся документы:

[!badge size="xl" variant="secondary" text="Завершение аренды"] - для завершения аренды у изначального контрагента;

[!badge size="xl" variant="secondary" text="Акт приема"] - оборудование автоматически перемещается на склад компании; 

[!badge size="xl" variant="secondary" text="Акт выполненных работ"] - в акте указывается сумма по услагам за аренду;

[!badge size="xl" variant="secondary" text="Оформление аренды"] - оформляется аренда на нового контрагента;

[!badge size="xl" variant="secondary" text="Акт передачи"] - оборудование автоматически переходит на нового контрагента;

[!badge size="xl" variant="secondary" text="Счет-фактура"] - указывается сумма по услагам за аренду.
