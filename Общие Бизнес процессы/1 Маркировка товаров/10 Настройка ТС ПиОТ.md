# Настройка ТС ПиОТ

## Описание

[!badge variant="light" text="ТС ПиОТ"] — это программные и программно-аппаратные решения, работающие в связке с контрольно-кассовой техникой. 
Их задача — собирать и передавать данные о маркированных товарах, подлежащих обязательной идентификации.

Основные функции [!badge variant="light" text="ТС ПиОТ"]:

• Передача информации о состоянии работы в Государственную информационную систему мониторинга товаров (ГИС МТ);

• Контроль работоспособности компонентов кассовой зоны;

• Проверка корректности распознавания кодов маркировки;

• Обеспечение соблюдения требований законодательства при работе с маркированными товарами;

• Блокировка продажи товаров, запрещённых к реализации на основании данных системы маркировки.

## Что нужно для установки

1. Необходимо обновиться для совместимой с ПИОТ версией кассового оборудования. Для касса атол это версия 5.16 [!badge variant="light" text="5.16"];

2. Пройти регистрацию в Единая Сервисная Платформе (ЕСП), для этого понадобится электронная подпись собственника организации или Генерального директора.

В личном кабинете отразятся все ваши зарегистрированные кассы. Для каждой кассы нужно преобрести лицензию (впервые выдается на 14 месяцев, далее покупать каждый год).

![](\images\маркировка\марк3.jpg)

Покупка лицензии

![](\images\маркировка\марк4.jpg)

Как только будет куплена лицензия, нужно установить [!badge variant="light" text="Модуль ЕСП"]. 

Его можно установить с помощью инструкции самостоятельно: https://disk.yandex.com.am/d/IJX0U69dtMqjPg

!!! Обратите внимание

Драйвер для кассы нужно установить, который приложен в инструкции
!!!

Если самостоятельно установить не удалось, то вы можете обратиться за установкой в службу поддержки ЕСП. В стоимость лицензии входит установка и настройка [!badge variant="light" text="Модуля ЕСП"].

## Соединение Модуля ЕСП и 1с

# Минимальные требования

Для кассы:

- Драйвер не ниже:   10.10.8.23

- ККТ должна быть зарегистрирована на ФФД 1.2.
  
- Параметр [!badge variant="light" text="Торговля маркированными товарами"]  должен быть включен.

Для ЛМ ЧЗ 2.0:

- ЕСМ: версия 1.6 или выше 
- lm-controller: версия 1.6 или выше

# Настройка ККТ

Перейдите в раздел [!badge variant="primary" text="Работа с оборудованием"]->[!badge variant="success" text="Подключаемое оборудование"].

Разверните вложенный список [!badge variant="light" text="ККТ с передачей данных"], выберите необходимую ККТ.

![](/images/маркировка/пиот.gif)

# Обновление компоненты ДККТ

В строке [!badge variant="light" text="Драйвер оборудования"] нажмите на кнопку ![](\images\маркировка\пиот.jpg)

В окне Драйвера оборудования нажмите кнопку на кнопку [!badge variant="light" text="Обновить"], далее в окне обновления нажмите на кнопку [!badge variant="light" text="Обновить из файла"] и [!badge variant="light" text="Продолжить"].

Далее у вас откроется Проводник, выберите необходимую компоненету – для этого перейдите в [!badge variant="light" text="Program Files"] → [!badge variant="light" text="ATOL"] →
[!badge variant="light" text="Drivers10"] → [!badge variant="light" text="ККТ"] → [!badge variant="light" text="1Cv83"], нажать левой кнопкой мыши по архиву и нажать на кнопку [!badge variant="light" text="Открыть"]. Дождитесь сообщения об успешном обновлении драйвера.

![](/images/маркировка/пиот1.gif)

В окне настройки ККТ нажмите на кнопку [!badge variant="light" text="Параметры подключения"].

![](\images\маркировка\пиот1.jpg)

В настройках соединения необходимо найти поле [!badge variant="light" text="Порт сервера GRPC"], укажите порт 4041.

![](\images\маркировка\пиот2.jpg)

!!! Обратите внимание! 
Если поля [!badge variant="light" text="Порт сервера GRPC"] в настройках оборудования нет, значит компонента не была обновлена до нужной
версии. Минимальная версия компоненты драйвера ККТ Атол, поддерживающая взаимодействие с ТС ПИоТ –
10.10.8.20.!!!

Для сохранения изменений нажмите на кнопку [!badge variant="light" text="Записать и закрыть"].
