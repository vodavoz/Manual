# Настройка ТС ПиОТ

## Описание

[!badge variant="light" text="ТС ПиОТ"] — это программные и программно-аппаратные решения, работающие в связке с контрольно-кассовой техникой. 
Их задача — собирать и передавать данные о маркированных товарах, подлежащих обязательной идентификации.

Основные функции [!badge variant="light" text="ТС ПиОТ"]:

• Передача информации о состоянии работы в Государственную информационную систему мониторинга товаров (ГИС МТ);

• Контроль работоспособности компонентов кассовой зоны;

• Проверка корректности распознавания кодов маркировки;

• Обеспечение соблюдения требований законодательства при работе с маркированными товарами;

• Блокировка продажи товаров, запрещённых к реализации на основании данных системы маркировки.

## Что нужно для установки

1. Необходимо обновиться для совместимой с ПИОТ версией кассового оборудования. Для касса атол это версия 5.16 [!badge variant="light" text="5.16"];

2. Пройти регистрацию в Единая Сервисная Платформе (ЕСП), для этого понадобится электронная подпись собственника организации или Генерального директора.

В личном кабинете отразятся все ваши зарегистрированные кассы. Для каждой кассы нужно преобрести лицензию (впервые выдается на 14 месяцев, далее покупать каждый год).

![](\images\маркировка\марк3.jpg)

Покупка лицензии

![](\images\маркировка\марк4.jpg)

Как только будет куплена лицензия, нужно установить [!badge variant="light" text="Модуль ЕСП"]. 

Его можно установить с помощью инструкции самостоятельно: https://disk.yandex.com.am/d/IJX0U69dtMqjPg

!!! Обратите внимание

Драйвер для кассы нужно установить, который приложен в инструкции
!!!

Если самостоятельно установить не удалось, то вы можете обратиться за установкой в службу поддержки ЕСП. В стоимость лицензии входит установка и настройка [!badge variant="light" text="Модуля ЕСП"].
